
# Basic REST API with exress.js and mongodb

## Installation
```
npm install
```

## Start dev server
```
npm run dev
```

## Start server for preview
```
npm start
```


## API endpoints

get all data
```
GET /posts/ 
```
find data
```
GET /posts/:postId 
```
posting new data
```
POST /posts/
```
deleting data
```
DELETE /posts/:postId
```
updating data
```
PATCH /posts/:postId
```

## Database Model
```
title: {
  type: String,
  required: true,
},
description: {
  type: String,
  required: true,
},

```
